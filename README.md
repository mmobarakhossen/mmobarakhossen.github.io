# mmobarakhossen.github.io

Personal academic website of **Md Mobarak Hossen** — PhD Candidate in Economics, Monash University.

Live site: https://mmobarakhossen.github.io

Built with the [Academic Pages](https://github.com/academicpages/academicpages.github.io) Jekyll template and hosted on GitHub Pages.

## Editing content

| What | Where |
| --- | --- |
| Homepage / bio | `_pages/about.md` |
| Research & publications | `_publications/*.md` |
| Teaching | `_teaching/*.md` |
| CV page | `_pages/cv.md` |
| Downloadable CV PDF | `files/cv.pdf` |
| Site settings, name, links | `_config.yml` |
| Top navigation menu | `_data/navigation.yml` |
| Profile photo | replace `images/profile.png` |

After editing, commit and push to the `main` branch; GitHub Pages rebuilds the site automatically.
